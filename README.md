# VoxelVariants

VoxelVariants is a Minecraft Forge mod that expands survival gameplay
with new materials, recipes, and utility features.

## Features
- New materials: Sulfur, Salt, Lemon
- Custom crafting recipes (Gunpowder, Cow Spawn Egg)
- /fly command for Survival mode

## Minecraft Version
- Minecraft 1.20.1
- Forge

## Development Status
This mod is still in active development.
Features and balance may change.
